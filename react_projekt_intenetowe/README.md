# Katalog Książek - Zadanie 6 (React)

Aplikacja typu SPA do zarządzania kolekcją książek, zbudowana w czystym React.

## Funkcjonalności
- Dodawanie, edycja i usuwanie książek (z potwierdzeniem).
- Zmiana statusu książki.
- Wyszukiwanie, filtrowanie i sortowanie (z zapamiętywaniem ustawień w localStorage).
- Zapis danych w `localStorage` za pomocą własnego hooka `useLocalStorage`.
- Formularz z 4 typami pól (text, textarea, select, radio) oraz pełną walidacją.
- Tryb Jasny / Ciemny.
- Statystyki kolekcji.
- Brak zewnętrznych bibliotek UI i zarządzania stanem.

## Instrukcja uruchomienia
1. Upewnij się, że masz zainstalowane środowisko Node.js.
2. Otwórz terminal w folderze głównym projektu (tam gdzie znajduje się plik `package.json`).
3. Zainstaluj zależności wpisując: `npm install`
4. Uruchom aplikację wpisując: `npm run dev`
5. Otwórz w przeglądarce link wyświetlony w terminalu (zazwyczaj `http://localhost:5173/`).
